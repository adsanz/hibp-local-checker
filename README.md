# Have I been pwned local password lookup

This is a simple rust program that allows to check either a password or a hash against a local file
containing the haveibeenpwned password list.
More info on that here: https://github.com/HaveIBeenPwned/PwnedPasswordsDownloader

## Why?

Well, the file is well over 30 GB in size, which is a pain in the ass to work with. I noticed the hashes are ordered.
Which allows usage of a binary tree to make the search more efficient.

## Usage

```bash
passchecker <file-path> <password|hash>
```
